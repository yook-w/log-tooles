## log-tooles 是一个快速操作console.log的工具集

>🍭 如果您喜欢，希望您可以点个好评

#### 功能快捷键
| 方法 | mac快捷键 |window快捷键|
| --- | --------- |----------- |
| 输出log | cmd+shift+L | ctrl+shift+L|
| 搜索log | cmd+shift+B | ctrl+shift+B|



#### 搜索log配置： 
>默认使用Baidu搜素引擎，若您希望使用其他，插件业支持Google或者Bing为默认搜索引擎。可在setting.json中修改默认值。
